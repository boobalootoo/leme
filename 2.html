Using Leme

📍 Map Locations: Tap the map to pin where an event is. Use the 📍 button to jump to your current location.

🙋 Activity Requests: Use the "Activity Request" option if you want others to suggest ideas or link existing events for you.

🗳️ Interests: Choose a status. Note that selecting a new status will replace your previous choice for that event.

💬 Chat/Suggestions: Use the text box on any card to suggest alternative times, locations, or provide links.
